This app is a pilot project in the school district of Groß-Gerau and Main-Taunus-Kreis

This app provides easy access to Lanis/the Hessen school portal. The functions include:

    Display of the substitution plan
    Push notifications for the substitution plan
    Advanced filter functions of the substitution plan
    Access to the school calendar
    Support for ‘My lessons’ (interface for teachers in progress)
    Support for messages
    Timetable view (interface of the teachers in progress)
    Direct 1-click login to the Lanis website
    File storage (retrieval of files)
